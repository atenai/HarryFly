# HarryFly

「ハリーフライ」

（概要・テーマ）
この作品は、初めて３D技術を使い開発したゲームです。
このゲームを開発するにあたり、基礎的な３D技術の勉強になった作品でした。

(開発時期・期間)
HAL東京2年生時の2017年12月～2018年2月(3ヶ月)

（開発ツール）
・開発環境：Visual Studio2015
・言語：C
・API：DirectX9
・3Dデータ制作：blender

（チームメンバー）
プログラマー1人（自分）

（ゲームシステム）
制限時間内にUAVでゴールを目指すゲームです。
・赤いボンベを取ると時間が回復します。
・コインを取るとスコアーが増えます。
・キューブに当たるとゲームオーバーになります。
・時間制限が来るとゲームオーバーになります。
・赤い線に到達したらゲームクリアーになります。

(操作方法)
・AとDキーでUAVを左右移動させることができます。
・WとSキーでUAVを上下移動させることができます。
・スペースキーでUAVを加速させることができます。

(起動方法)
「ハリーフライ(ビルド)」というフォルダーを開いてください。
その中にある、「HarryFly.exe」というアプリケーションをダブルクリックしてください。
